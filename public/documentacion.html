Aplicación CRUD de Películas y Series

Descripción General

El proyecto es una aplicación web diseñada para gestionar una biblioteca de películas y series, permitiendo a los usuarios crear, leer, actualizar y eliminar información sobre películas. Además, incluye características como la carga y gestión de imágenes, filtrado por géneros y búsqueda de películas.

Tecnologías y Librerías Utilizadas

Frontend

  • HTML5: Estructura básica de la aplicación.
  • CSS3: Estilos para la presentación visual.
  • JavaScript (ES6+): Lógica del cliente para interactividad.
  • Bootstrap: Framework CSS utilizado para el diseño responsivo y componentes estilizados.

Backend

  • Node.js: Entorno de ejecución para JavaScript en el servidor.
  • Express.js: Framework para construir aplicaciones web y APIs en Node.js.
  • MySQL: Sistema de gestión de bases de datos relacional utilizado para almacenar la información de las películas y series.

Servicios

  • Cloudinary: Servicio de almacenamiento en la nube utilizado para gestionar y optimizar imágenes.
  • dotenv: Paquete para manejar variables de entorno.

Middleware

  • body-parser: Middleware para analizar los cuerpos de las solicitudes HTTP.
  • cors: Middleware para habilitar CORS (Cross-Origin Resource Sharing), permitiendo que la API sea accesible desde diferentes dominios.
  • morgan: Middleware para registrar las solicitudes HTTP en la consola, útil para el desarrollo y depuración.
  • multer: Middleware para manejar la carga de archivos, utilizado para subir imágenes y banners de películas y series.

Estructura del Proyecto

movie-crud/
├── api/                # Controladores y lógica de negocio
├── config/             # Configuraciones (DB, Cloudinary)
├── public/             # Archivos estáticos
│   ├── css/            # Estilos
│   ├── js/             # JavaScript del cliente
│   └── carruselimg/    # Imágenes del carrusel
├── routes/             # Rutas de la API
└── server.js           # Punto de entrada

Base de Datos

MySQL: Utilizado como base de datos para almacenar la información de las películas. La estructura de la tabla peliculas incluye campos como id, titulo, contenido, categoria, anio, genero, imageUrl, bannerUrl, y watchUrl.

Alojamiento

Railway: El proyecto está alojado en Railway, una plataforma que permite implementar aplicaciones de manera sencilla y rápida.
